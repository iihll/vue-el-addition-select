## Usage
`yarn add vue-el-addition-select`
- global
``` js
import ELAdditionSelect from './ELAdditionSelect.vue'
Vue.use(ELAdditionSelect)
```


- component
``` js
import { ELAdditionSelect } from 'el-input-tag'

export default {
    components: { ELAdditionSelect }
}
```
